# IBM Global AI Racing League

This repository contains the source code for an autonomous Artificial Intelligence agent developed for the IBM Global AI Racing Competition. The vehicle is trained to navigate the "Corkscrew" track using The Open Racing Car Simulator (TORCS).

## Technology Stack
* **Language:** Python 3
* **AI Algorithm:** Soft Actor-Critic (SAC) via Stable Baselines 3
* **Simulator:** TORCS (The Open Racing Car Simulator)
* **Code Assistant:** IBM Granite Models
